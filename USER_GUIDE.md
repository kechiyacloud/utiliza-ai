## Getting Started

### Logging In

1. Open the application in your browser (provided by your administrator)
2. Enter your **Email ID** and **Password**
3. Click **Sign In**

> **Tip:** Use the eye icon next to the password field to show or hide your password while typing.

If you see an error message after clicking Sign In, double-check your email and password. Contact your administrator if you cannot log in.

---

### Navigating the App

After logging in, you will land on the **Dashboard**. The main navigation is on the left sidebar:

| Sidebar Item | Where it takes you |
|-------------|-------------------|
| Dashboard | Executive overview with KPIs and charts |
| Employees | Employee master list and management |
| Projects | Project list and details |
| Allocation | Resource utilization dashboard |
| Availability | Employee availability timeline |
| Clients | Client and partner management |
| Organization | Org chart and department structure |
| Analytics | Advanced workforce analytics |
| Settings | Your profile and preferences |

Click any item to switch pages. Your current page is highlighted in the sidebar.

---

## Dashboard

The dashboard is your **command center** — it shows you the health of the entire organization at a glance.

### What you see

**KPI Cards (top row)**

| Card | What it means |
|------|--------------|
| Company Utilization | % of employees currently allocated to projects. Target is 85%. |
| Billable Headcount | Number of employees working on billable projects right now. |
| Bench Headcount | Number of employees with no current project assignment. |
| Upcoming Bench (30d) | Employees whose project ends within the next 30 days. |

Click any KPI card to jump to the detailed allocation page.

**Allocation Forecast Chart**

A bar chart showing the next 3 months:
- **Blue bars** = Available capacity (unallocated FTE)
- **Orange bars** = Currently allocated FTE

Use this to spot months where you'll be over or under capacity.

**Workforce Breakdown Chart**

Shows how your headcount is split:
- Billable · Internal · Bench · Notice Period

**Projects at Risk**

Lists projects flagged as high-risk. Review these regularly.

**Todo List**

Your personal action items. To use it:
1. Type a task in the **"Add a new task..."** box
2. Click the **Add** button (turns active when you type)
3. Check the checkbox when a task is complete
4. Click the trash icon to delete a task

### Filtering by Department

Use the **Department dropdown** at the top right to filter all dashboard metrics to a specific team.

### Quick Actions from the Dashboard

| Button | What it does |
|--------|-------------|
| Add Employee | Opens the Add Employee wizard |
| Add Project | Opens the Add Project panel |
| Add Client | Opens the Add Client modal |

## Employee Management

Go to **Employees** in the left sidebar.

### Viewing Employees

The employee table shows all active employees. Each row displays:
- **Name and Employee ID**
- **Department** and **Location**
- **Designation** (job title)
- **Status** (Billable / Bench / Notice Period / etc.)
- **Skills** (tags)

Click any row to open the **Employee Profile** with full details.

### Filtering & Searching Employees

**Quick Department Filter (chips)**

Above the table, there is a row of department chips. Click a department name to filter the table. Click **All** to reset.

**Search Bar**

Type anything in the search box — it searches across name, ID, designation, location, department, and skills simultaneously.

**Advanced Filter Panel**

Click the **Filter** button to open the full filter panel. You can filter by:

| Filter | Options |
|--------|---------|
| Department | Multi-select: any combination of departments |
| Employee Type | Full Time, Intern, Consultant, Contract |
| Skills | Multi-select with search |
| Location | Multi-select |
| Status Tags | Billable, Bench, Notice Period, etc. |
| Designation | Multi-select |

After selecting your filters, click **Apply**. To start over, click **Clear All**.

---

### Adding an Employee

Click the **Add Employee** button (user-plus icon) in the top right, or use the dashboard quick action.

This opens a **4-step wizard:**

---

**Step 1 — Personal Information**

| Field | Notes |
|-------|-------|
| Employee ID | Auto-generated or enter manually |
| Full Name | Required |
| Email | Required, must be a valid email format |
| Phone Number | Required |
| Date of Joining | Required — determines "New Joiner" status |
| Location | Select from dropdown |
| Work Mode | Office / Remote / Hybrid |
| Profile Photo | Upload image (max 2 MB). Preview shows immediately. |

Click **Next** when done. The form validates before moving forward.

---

**Step 2 — Professional Details**

| Field | Notes |
|-------|-------|
| Designation | Job title (e.g., Senior Engineer) |
| Department | Select from dropdown |
| Reporting Manager | Search and select from existing employees |
| Employment Type | Full Time / Intern / Consultant / Contract |
| Skills | Search and select multiple skills. Suggestions appear based on department. |
| Certificates | Add any certifications with issue/expiry dates and file upload |

> **Note:** Status and Allocation % are **read-only** — they are automatically calculated from the projects you assign in Step 3.

---

**Step 3 — Project Assignments**

To assign the employee to a project:
1. Search and select a **Project** from the dropdown
2. Enter their **Role** in the project
3. Set the **Allocation %** (0–100)
4. Set **Start Date** and optionally an **End Date**
5. Click **Add**

The project appears in the assignments table below. You can delete any row using the trash icon.

Repeat for multiple projects. The **Status** and **Allocation %** at the top will update automatically.

---

**Step 4 — Preview**

Review all entered information in a read-only summary. If anything needs correcting, click **Back** to return to that step.

When satisfied, click **Confirm** to save the employee.

---

### Editing an Employee

1. Find the employee in the table
2. Click the **pencil (edit) icon** on their row, or open their profile and click **Edit**
3. The same 4-step wizard opens, pre-filled with existing data
4. Make your changes and click **Confirm**

---

### Deleting an Employee

1. Click the **trash icon** on the employee's row
2. A confirmation dialog appears — click **Confirm** to delete
3. This action cannot be undone

---

### Bulk Importing Employees

Use this when onboarding multiple employees at once.

1. Click the **Bulk Actions** dropdown → **Import Employees**
2. Download the template first if you haven't already (**Download Template**)
3. Fill in the template with employee data
4. Drag and drop the file into the upload area, or click to browse
5. Accepted formats: **CSV, Excel (.xlsx), JSON**
6. The system shows a preview — check for validation errors
7. Click **Confirm** to import

> **Tip:** Use the **Download Template** option to ensure your columns match exactly what the system expects.

---

### Exporting the Employee List

1. Apply any filters or search terms to narrow the list
2. Click **Bulk Actions** → **Export to CSV**
3. The file downloads immediately to your browser's download folder

## Project Management

Go to **Projects** in the left sidebar.

### Viewing Projects

The projects page has two sections:

**Overview cards (top):**
- Active Projects / Completed / At-Risk / On-Track — click any card to filter the list below

**Projects list (below):**
Each row shows project name, client, status, start and end dates, team size, and budget.
- Click a row to open the full **Project Details** page
- Status badges are color-coded: Green (Active), Gray (Completed), Red (At-Risk), Blue (On-Track)

---

### Adding a Project

Click **Add Project** (from the dashboard or Projects page). A panel slides in from the right.

---

**Section 1 — Project Basics**

| Field | Notes |
|-------|-------|
| Project Name | Required |
| Project Type | Client / Internal / Partner / POC |
| Client Type | Appears for Client type: Direct Client or Partner Client |
| Client / Partner | Select from dropdown. You can add a new client/partner inline (see below). |
| Project Status | Not Started / In Progress / On Hold / Completed |
| Billable | Billable / Non-Billable / Shadow |
| Start Date | Required |
| End Date | Optional |

**Adding a new Client or Partner inline:**
If the client doesn't exist yet, click the **+ Add** button next to the client dropdown. A small modal appears where you enter the name and confirm. The new client is immediately available in the dropdown.

---

**Section 2 — Team Members**

To add a person to the project:

1. Search for an **Employee** in the search box
2. Their **Role** auto-fills from their profile (you can change it)
3. Set **Allocation %** (0–100)
4. Set **Allocation Start Date** (and optionally an End Date)
5. Choose **Billable** or **Shadow**
6. Enter weekly hours in **W1, W2, W3, W4** fields
7. Click **Add to Team**

The team member appears in the table below. Click the trash icon to remove them.

> **Weekly hours explained:**
> W1 = first week of their allocation period, W2 = second week, etc.
> These are used for capacity planning and export reports.

---

When the form is complete, click **Save Project**.

---

### Editing a Project

1. Click the project row to open Project Details
2. Click **Edit** at the top
3. Update any fields — you can edit the name, status, dates, team members, commercials, and scope
4. Click **Save**

**Editing commercials (budget, billing type):**
In the Project Details page, scroll to the **Commercials** section and click **Edit** to update budget, billing type, contract type, and revenue model.

**Editing scope (objectives, deliverables):**
In the **Scope** section, click **Edit** to update objectives, deliverables, milestones, and timeline notes.

---

### Managing Project Resources

Inside a Project Details page, the **Resources** tab shows all team members.

| Action | How to do it |
|--------|-------------|
| Add a new resource | Click **Add Resource**, fill the form, save |
| Update allocation % | Click the edit icon on the resource row |
| Change dates or role | Click the edit icon on the resource row |
| Remove a resource | Click the delete icon on the resource row |

---

### Exporting Project Data

From the Project Details page:

| Export | How |
|--------|-----|
| PDF report | Click **Export PDF** — downloads a formatted report of all team members and allocations |
| CSV | Click **Export CSV** |
| Excel | Click **Export Excel** |

## Resource Allocation

Go to **Allocation** in the left sidebar.

This page gives you an organization-wide view of how your people are being used.

### Reading the Allocation Dashboard

**6 Metric Cards at the top:**

| Card | What it means |
|------|--------------|
| Total Resources | Total headcount tracked in the system |
| Billable | Employees on billable project work |
| Non-Billable | Employees on internal/non-billable work |
| Bench | Employees with no current assignment |
| Avg Utilization | Average allocation % across all employees |
| Overallocated | Employees assigned to more than 100% combined allocation |

**Filtering the dashboard:**

Use the dropdowns at the top to filter by:
- Department
- Location
- Resource Type

---

### Exploring Department & Project Utilization

**Project Utilization Panel (left)**

Lists all projects sorted by how many resources they're using. Click any project to see a breakdown of everyone assigned to it.

**Department Breakdown Chart (right)**

Shows a chart of Billable vs Non-Billable headcount per department. Helps identify which teams are most/least utilized.

**Employee Allocation Drill-down**

After clicking a project in the left panel, the right side switches to show:
- Each assigned employee
- Their role and allocation %
- Their billable/shadow tag

Click elsewhere or the **X** button to go back to the department chart.

---

### Bench Forecast & Project Matching

**Forecast Bench section (lower panel):**

This list shows employees whose current allocation **ends within the next 30 days** — these people are about to become available.

For each employee it shows:
- Their name and current project
- The date their allocation ends
- Their key skills

**Finding a project for a bench employee:**

1. Click on any employee in the Forecast Bench list
2. A **Possible Project Matches** panel opens on the right
3. It shows projects where the employee's skills are a good fit
4. Each suggestion shows project name, client, status, and available allocation %
5. Use this to plan re-allocation before the employee becomes idle

---

### Bulk Importing Allocations

If you manage allocations in a spreadsheet, you can upload them directly.

1. Click **Import Allocations**
2. Upload your CSV or Excel file
3. Review the preview — check for warnings or errors
4. Toggle **Dry Run** to test without saving
5. Click **Confirm** to commit the import

## Availability Timeline

Go to **Availability** in the left sidebar.

This page shows a **Gantt-style timeline** of every employee's allocations across time.

### Reading the Timeline

- Each **row** is one employee (or one allocation record)
- **Columns** are weeks grouped by month
- **Colored bars** show active project assignments — each project has its own color
- Empty cells mean the employee is **unallocated (bench)** for that period
- The current week is **highlighted**
- The view auto-scrolls to today's date when you open the page

### Filtering the Timeline

Click the **Filter** button to open the filter panel:

| Filter | What it does |
|--------|-------------|
| Department | Show only employees from selected department(s) |
| Location | Show only employees in selected location(s) |
| Project | Show only employees on selected project(s) |
| Start Month | Set the left edge of the visible date range |
| End Month | Set the right edge of the visible date range |

The filter panel also shows a **preview summary** before you apply:
- How many rows match
- How many are allocated vs bench

Click **Apply Filters** to update the view. Click **Clear All** to reset.

### Exporting the Timeline

Use the **Export** dropdown:

| Option | What you get |
|--------|-------------|
| Export to Excel | Spreadsheet of all visible timeline data |
| Export to PDF | Formatted PDF of the timeline |

### Viewing Clients

- **Left panel:** Scrollable list of all clients (with a search bar at the top)

### Adding a Client

1. Click **Add New Client**
2. Fill in the form:

| Field | Notes |
|-------|-------|
| Client Name | Required |
| Industry | Dropdown (Retail, Finance, Healthcare, etc.) |

1. Select the client from the left list
2. Click the **Edit** button in the detail panel

1. Select the client
2. Click **Delete**

Partners are companies that refer clients to you. They are managed from within the **Add/Edit Project** panel (when project type is "Client" and client type is "Partner Client"), as well as from the Clients page.

To add a partner:
1. Go to the Clients page → click **Partners** (or access via Add Project panel)
2. Click **Add Partner**, enter the partner name, and save

## Tips & Best Practices

### Keeping Data Accurate

- Always set an **Allocation End Date** when adding a team member to a project. This keeps the Forecast Bench list accurate.
- Update employee **Status** by managing their project assignments — do not manually override it.
- When an employee leaves a project early, update the end date in the project's Resources section immediately.

### Staying on Top of the Bench

- Check the **Upcoming Bench (30-day)** card on the dashboard every Monday
- Use the **Bench Forecast** section in Allocation to proactively find project matches before employees become idle
- Check the **Skill Gap Analysis** in Analytics to match bench employees to understaffed projects

### Onboarding New Employees

- Use **Bulk Import** for groups of 5+ employees to save time
- Assign them to projects in **Step 3** of the Add Employee wizard so their status is Allocated from day one
- Make sure to add their **skills** accurately — this powers the project matching feature

### Exporting for Reporting

| Report | Best export format |
|--------|------------------|
| Headcount / employee list | CSV (easy to open in Excel) |
| Project resource plan | PDF (shareable, formatted) |
| Availability schedule | Excel (easy to filter and sort) |
| Analytics / workforce health | PDF (for leadership presentations) |
