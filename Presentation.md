# Project Dashboard: A Centralized Organizational Platform

## What is this Application?
Project Dashboard is a comprehensive, centralized organizational platform designed to manage employees, projects, and resource allocations from a single interface. It provides a complete organizational view with role-based access control, catering to the needs of individual employees, project managers, and organizational leadership.

## The Problem Statement
Organizations often struggle with fragmented data spread across multiple tools, making it difficult to maintain a clear overview of resources, project statuses, and employee allocations. This fragmentation leads to:
*   Inefficient resource allocation.
*   Lack of visibility into project health and bench strength.
*   Difficulty in tracking employee skills and availability.
*   Challenges in generating consolidated reports for decision-making.

## What Problems is it Solving?
The Project Dashboard addresses these challenges by:
*   **Centralizing Information:** Consolidating employee, project, and allocation data into a single source of truth.
*   **Enhancing Visibility:** Providing role-specific dashboards that offer clear insights into utilization, bench risk, and project progress.
*   **Streamlining Operations:** Enabling efficient project allocation management and tracking of resource availability over time.
*   **Empowering Decision Making:** Offering robust analytics and exportable reports (CSV, Excel, PDF) to support strategic organizational decisions.

## How it Solves these Problems (Key Features)

### 📊 Comprehensive Dashboard & Analytics
*   **Summary Cards:** Instant visibility into Total Employees, Clients, Projects, and Bench Strength.
*   **Workforce Pulse:** Identifies bench risks, tracks transitions, and highlights skill gaps.
*   **Organization Insights:** Org chart and departmental breakdowns for structural clarity.

### 👥 Employee Management
*   **Centralized Master List:** Searchable and filterable employee directory.
*   **Skill & Certification Tracking:** Detailed tracking of employee capabilities.
*   **Bulk Operations:** Support for bulk upload via Excel/CSV for easy onboarding.
*   **Allocation History:** Historical tracking of per-employee allocations.

### 💼 Project Management
*   **Detailed Project Views:** Track project status, type, and associated clients.
*   **Team Allocations:** Manage and view weekly resource allocations (W1–W4) per project.
*   **Client Management:** Manage both internal clients and partner/vendor clients, including budget tracking.

### 📈 Allocation & Resource Availability
*   **Utilization Metrics:** Track Billable, Internal, and Bench resource metrics.
*   **Timeline Views:** Visual timelines of employee availability for precise forecasting.
*   **Project Matching:** Tools to align available resources with upcoming project needs.

### ⚙️ System Features
*   **Role-Based Access:** Secure JWT-based authentication ensuring users only see what they should.
*   **Export Capabilities:** Data from all major pages can be exported to CSV, Excel, or PDF.
*   **Feedback Mechanism:** Built-in system for users to submit feedback, bug reports, or feature requests.
